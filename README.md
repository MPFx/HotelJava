# Sistema de Hotel (Java)

## Versión
**1.0**

## Descripción
Sistema de gestión hotelera desarrollado en Java. Permite gestionar clientes, empleados, habitaciones, reservas, servicios adicionales y facturación.

## Alcance

### ✅ Qué hace
- Registrar clientes, empleados, habitaciones y servicios
- Gestionar reservas (crear, confirmar, activar, cancelar, finalizar)
- Control de disponibilidad de habitaciones
- Tipos de habitación: Simple, Doble, Suite
- Servicios adicionales (Spa, Restaurante, Lavandería, Transporte)
- Facturación con IVA (19%)
- Estados de reserva: PENDIENTE, CONFIRMADA, ACTIVA, FINALIZADA, CANCELADA

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real
